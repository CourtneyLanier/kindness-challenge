Slack Kindness Challenge with Rhoda

Run a seasonal kindness campaign right inside Slack (e.g., 100 Acts in 100 Days). Anonymous or named submissions, optional prayer requests, and an in-channel candle progress bar.

  • Add Rhoda to your workspace.
  • Create/choose a channel and add the app.
  • Run /kindness-config to set Start/End/Goal.
  • Teammates use /kindness. Reset anytime with /kindness-reset.
